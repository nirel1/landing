01. Why Move?

The best smart contract language and revolutionary VM

Learn more

Security

Move provides enhanced security through its deterministic and statically typed language, dramatically reducing memory-related vulnerabilities.

  • Memory safety
  • Native scarcity controls
  • Bytecode verification

Speed

Move is designed for high-performance - with parallel execution and direct compilation to bytecode - eliminating interpreter overhead for faster runtime performance.

  • Parallel execution capability
  • No interpreter overhead
  • Direct bytecode compilation

Compatibility

Move is designed to be cross-platform. It follows an embedded language design, making it lightweight and dependent on the underlying blockchain for chain-specific implementations.

  • Cross-platform language
  • Embedded language design
  • Supports multiple blockchains
